# Zsh

- https://ohmyz.sh/ extension
- tons of aliases - run `alias`
    - `cd` not required
    - `d` - last visited dirs
    - `..`, `....` - move up
    - `/` - root folder
    - `-` - previous folder
    - `l`, `ll`, `la`
- recursive path expansion `cd /h/jv/Id`
- better autocompletion
- up/down for quick search
- plugins (git, docker, ...)
- themes
