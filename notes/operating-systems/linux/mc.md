# Midnight Commander

## Sources
http://www.trembath.co.za/mctutorial.html#anchor16

- `F10` - exit
- `tab` - move between tabs
- `insert` - select multiple files
- `ctrl + u` - switch tabs
- `ctrl + o` - open shell
- `ctrl + r` - refresh
- `ctrl + s` - type and jump in this folder
- `ctrl + \` - bookmarks
- `alt + c` - quick cd
- `alt + h` - show history
- `alt + t` - cycle through different views
- `alt + .` - toggle hidden files
- `alt + ?` - global search

---
- `ctrl + x, d` - compare
- `ctrl + x, c` - chmod

## File selection
- `+` - select files
- `\` - unselect files
- `*` - reverse selection
